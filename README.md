# Campus Placement Analytics Using MySQL and Power BI

## Project Overview

This project demonstrates an end-to-end campus placement analytics solution using MySQL and Power BI.

The project focuses on:

- Relational database design
- Data cleaning
- Data validation
- Database normalization
- Executive dashboard development
- Business intelligence reporting

---

## Tech Stack

- MySQL
- Power BI
- Data Modeling
- Business Intelligence

## Database Architecture

Tables:

- Enrolments
- Candidates
- Applications
- Interviews
- Job Openings

Relationships were created using primary and foreign keys to ensure referential integrity and eliminate redundancy.

## Executive Insights

- Interview cancellation rate exceeded 55%.
- Social Media roles generated the highest demand.
- Placement conversion remains low.
- Candidate screening and interview management require optimization.
